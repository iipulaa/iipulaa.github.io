# Web Kehitys

Kansiossa kuvia sovelluksista, jotka olen tehnyt React JavaScript kirjaston avulla.
"Movies" sovellus haki tietokannasta ohjelmistossa olevia elokuvia ja niiden tietoja. Tiedoille muokattiin visuaalisesti miellyttävä ja selkeä käyttöliittymä. Klikkaamalla sai elokuvan trailerin näkyviin.
